# 🧮 Calculator

A simple Calculator application built using **React**, **Vite**, **Tailwind CSS**, and **shadcn/ui**. It performs basic arithmetic operations with a clean and responsive user interface.

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear button
* 📱 Responsive Design

---

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* shadcn/ui
* Lucide React

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/calculator.git
```

Go to the project folder

```bash
cd calculator
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

---

## 📦 Build

```bash
npm run build
```
